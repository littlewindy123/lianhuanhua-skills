# Line dog example

Place a simple cartoon dog reference image at:

```text
input/character.png
```

Then ask Codex:

```text
Use $lianhuanhua with examples/line-dog/input/story.txt and examples/line-dog/input/character.png. Create a warm, restrained 9:16 comic video.
```

The reference image is intentionally not bundled so the repository does not redistribute third-party artwork.
